# Space2dot
## What is space2dot
Change the space into dot.

## Installation
Install it yourself as:

    $ gem install space2dot

## Usage

From the terminal:

    $ space2dot style sheet
    => .style.sheet # Output to the standard output

## Shorthand

A shorthand alias for space2dot can also be used.

```
s2. new branch
```
